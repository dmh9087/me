# 个人网页设计文档

**日期**: 2026-05-30  
**主题**: 创意手绘风格个人网页 — 电子信息专业

---

## 一、概述

打造具有纸质手绘质感的单页个人网站，融入电路板、LED、导线等电子信息专业元素，风格童趣又贴合专业属性。采用单页长滚动 + 固定导航布局，纯前端 HTML/CSS/JS 实现。

---

## 二、文件结构

```
/
├── index.html      — HTML 结构（内联 Google Fonts）
├── styles.css      — 全部样式
├── script.js       — 全部交互逻辑
```

零依赖，零构建工具，浏览器直接打开即可运行。

---

## 三、视觉风格

### 3.1 色彩
- **底色**: 暖米色笔记本纸页 `#fdf6e3`
- **主文字**: 手绘墨水蓝黑 `#2c3e50`
- **电路点缀**: 铜绿 `#5a8f7b`、暗金 `#c2a44a`
- **LED**: 红 `#ff4444`、绿 `#44ff44`、黄 `#ffaa00` 闪烁

### 3.2 字体 (Google Fonts)
- 标题: **Caveat** — 手写风格
- 正文: **Neucha** — 略带不规则衬线
- 装饰: **Gaegu** — 童趣手绘感（用于标签等小元素）

### 3.3 边框与线条
- 使用 `box-shadow` 多重偏移 + `border-radius` 不规则值模拟手绘边框
- 装饰线用 `border-bottom: 2px dashed` 模拟铅笔线条

## 四、页面板块（从上到下，固定导航栏在顶部）

### 4.1 导航栏
- 固定在页面顶部，半透纸张背景
- 标签: 首页 | 关于 | 技术栈 | 项目 | 留言墙 | 音乐角 | 联系
- 点击平滑滚动到对应板块

### 4.2 Hero 首屏
- 大标题（名字）+ 副标题（一句话介绍）
- **打字机效果**: JS 逐字输出副标题，带闪烁光标
- **鼠标光点**: Canvas 层覆盖，粒子跟随鼠标

### 4.3 关于我
- 手绘便签纸卡片
- 一段自我简介文字
- 教育经历/专业背景

### 4.4 技术栈
- 技能以手绘"标签云"排列
- 每个标签外框略微歪斜（随机 rotate ±2°）
- 标签颜色随机从配色方案中选取

### 4.5 项目案例
- 卡片网格布局
- 每张卡片模拟贴在本子上的纸条，顶部带"图钉"装饰 (CSS 圆形 + 阴影)
- 项目名、简述、链接

### 4.6 留言墙
- 表单：昵称 + 留言内容 + 提交按钮（手绘风格）
- 已发布的留言渲染为便利贴，随机微旋转 ±3°
- 数据存储在 `localStorage`
- 支持删除自己的留言

### 4.7 音乐角
- 外观模拟手绘收音机
- 输入框粘贴外部音频 URL
- 创建 `<audio>` 元素播放
- 播放列表存储在 `localStorage`

### 4.8 页脚
- 联系方式 + 社交链接
- 手绘信封图标（CSS 绘制）

## 五、动效与互动

### 5.1 背景电路板
- 页面背景随机分布 SVG/CSS 手绘电路走线
- 横向/纵向导线 + 转折节点

### 5.2 LED 闪烁
- 散布在电路走线上的小圆点
- CSS `@keyframes` 控制间歇闪烁（不同节奏）

### 5.3 导线流动动画
- 虚线 `stroke-dasharray` + `stroke-dashoffset` 动画
- 模拟电流在导线上流动

### 5.4 鼠标跟随光点
- Canvas 画布，粒子系统
- 粒子在鼠标周围生成，逐渐消散
- 暖黄色小光点

### 5.5 打字机效果
- Hero 区副标题逐字出现
- 末尾闪烁光标块
